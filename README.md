## Directory Structure


## Hosting

本仓库使用 GitHub Pages 托管隐私政策页面。

启用方式：
1. 进入仓库 Settings → Pages
2. Source 选择 "Deploy from a branch"
3. Branch 选择 "main" + "/ (root)"
4. 保存后访问：https://yangfuhe.github.io/privacy/
